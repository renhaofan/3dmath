◆ computeProjectiveInverse()

Matrix4 Matrix4::computeProjectiveInverse ( ) const

computeProjectiveInverse

A, B, C, D are 2x2 matrix blocks and A is invertible matrix.

(. denotes place-holder)

[ A | B ]

[ ..+.. ]

[ C | D ]

det = |A| * | D - C * A^{-1} * B|, inverse matrix is as follows

A' = A^-1 - (A^-1 * B) * C'

B' = -(A^-1 * B) * D'

C' = -D' * (C * A^-1)

D' = (D - ((C * A^-1) * B))^-1

[ A' | B' ]

[ ...+.... ]

[ C' | D' ]

Warning
The matrix is invertable even if det(A)=0, check det(A) before calling.
Exceptions
Nota projective matrix.
Projectivematrix is not reversible.
Returns
result
